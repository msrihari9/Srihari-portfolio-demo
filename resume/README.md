# Resume folder

- `resume.html` — a styled, print-friendly résumé page. Open it and click **Print / Save as PDF**
  (or use your browser's Print dialog) to export a polished `Srihari-M-Resume.pdf`.
- Once you export a PDF, drop it in this folder (e.g. `Srihari-M-Resume.pdf`) and update the
  "View / Download Resume" link in `../index.html` to point at it directly if you'd rather serve
  a PDF than the HTML version.
